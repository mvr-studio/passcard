# AddressQrCode

A component to present user's address as a QR code.

```tsx
import { AddressQrCode } from '@passcard/ui'
```

## Playground

import { ExampleAddressQrCode } from '../../../../components/ExampleAddressQrCode'

<ExampleAddressQrCode />

## API

| Property | Type     | Default | Description                 |
| -------- | -------- | ------- | --------------------------- |
| address  | `string` |         | Address to display as a QR. |
